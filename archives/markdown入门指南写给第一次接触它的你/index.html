Markdown——入门指南：写给第一次接触它的你

2025年04月18日
阅读时长: 1 分钟

这篇文章是写给 Markdown 零基础的新手的。不需要你有代码基础，只要你会打字，就可以轻松上手。

为什么选择 Markdown？

Markdown 不是一种软件，而是一种轻量级标记语言，也就是说，你用一些非常简单的"标记符号"就能控制文章的标题、加粗、引用、列表等格式。非常适合：

  • 技术博客写作
  • 产品文档撰写
  • 写作笔记 / 读书摘录
  • 编写简历、GitHub 项目说明等

Markdown VS 记事本 VS Word

功能 / 工具记事本WordMarkdown
格式化能力❌ 几乎没有✅ 强大但复杂✅ 简单可控
写作专注度✅ 纯文本❌ 功能过多✅ 极简风格
可移植性✅ 高❌ 格式兼容性差✅ 极高
技术场景适配❌ 不适用❌ 不适用✅ 非常适合
学习成本✅ 极低❌ 略高✅ 低，10分钟掌握
